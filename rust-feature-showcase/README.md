# rust-feature-showcase

一个放在公开仓库根目录下的综合 Rust 示例程序。

## 覆盖点

- 命令行解析
- 数据建模与序列化
- trait 与 trait object
- 泛型工具函数
- 线程并发
- 自定义错误处理
- 单元测试

## 命令示例

```powershell
cargo run -- print-sample
cargo run -- validate --input sample-data.json
cargo run -- analyze --input sample-data.json
cargo run -- analyze --input sample-data.json --format json
```
